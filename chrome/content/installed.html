Clear Fields 4.0.3 for Firefox
Thank you for installing Clear Fields!
You are using version 4.1.0.
To add the Clear Fields buttons to your toolbar, right click on it and select Customize... then drag the buttons to the toolbar.
If you want this done automatically, open the Tools menu -> Clear Fields, and select Add Main Toolbar Buttons.
The in-field clear buttons are enabled by default since version 4.0.0. You may disable them in the Tools menu.
If you can't find the Tools menu, open the Main Menu, click Preferences and select the Menu Bar.
